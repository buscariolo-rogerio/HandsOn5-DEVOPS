## HandsOn5-DEVOPS 
# A quinta lição de DEVOPS, baseada em code review
Fizemos o projeto com o objetivo principal de estudar e ver o funcionamento do code review na prática.
Dentro do projeto temos um site com 5 links, cada link leva para uma página diferente. Cada página tem informações sobre países que nós alunos queremos visitar um dia
Países como:
- Suiça
- Grécia
- Tasmânia
- Estados Unidos
- Islândia
  
Cada país com curiosidades sobre eles em cada página.
